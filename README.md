# Tips Prediction using Linear Regression

## Project Overview

This project is part of my learning journey in **Machine Learning and Data Analysis**.
The goal of this project is to build a simple regression model to predict restaurant tips based on available features in the dataset.

As a learner in Machine Learning, I am developing practical skills in data preprocessing, model building, and evaluation using real datasets.

---

## Dataset

The project uses the well-known **Tips Dataset**, which contains information about restaurant bills and tipping behavior.

Main features include:

* Total bill
* Number of people
* Gender
* Smoking status
* Day and time of visit
* Tip amount

---

## Objective

The main objective of this project is to:

* Understand regression problems
* Apply Linear Regression
* Practice data preprocessing
* Evaluate model performance
* Improve practical Machine Learning skills

---

## Methodology

The following steps were performed:

1. Data exploration and understanding

5. Model evaluation
